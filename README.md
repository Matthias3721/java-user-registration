# Java User Registration & Login (Console)

Simple Java console application demonstrating basic backend concepts:
- OOP (User, Service layer)
- User registration and login
- In-memory storage using Map
- Basic validation logic

## Features
- Register user (email uniqueness, age validation)
- Login user by email
- Clear separation of concerns (Main / Service / Model)

## Technologies
- Java
- Collections (Map)

## Purpose
Learning backend fundamentals and clean code structure.
